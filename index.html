Ever since Betheme was just an idea, we knew that it would be different from all other multipurpose WordPress themes we’d tried before.

We wanted to build a theme that could easily adapt to any project you need to work on without writing any code. A theme designed from scratch to save you time & help you enjoy your freedom...

Modify and adapt websites or build from scratch

Build complex layouts quickly with the Muffin Builder

The Muffin Builder allows you to build any page layout using Sections and containers called Wraps. You can choose from 200+ shortcodes and various styling options. Create more advanced and good looking content for your website.

Customize your website as you want using Theme Options

Use Theme Options to globally set the look of Blog, Portfolio, and Shop pages on your website. Built-in 500+ settings allow you to modify colors, fonts, grid size, layouts, and extras such as SEO, social media, sidebars, translations, and footer.

Create headers using pre-defined layouts or the Header Builder

Choose from 16 pre-defined header layouts or use the Header Builder plugin to make any header you want. You can create different menus for each page. You can also select a custom menu just for mobile devices.

One-click website installation

Fully customizable

Each website built with Betheme will look in a different manner. What you need to create amazing websites with no coding knowledge is the Muffin Builder paired with the options panel.

Speed optimization

The theme is fully optimized. Attention to detail, well-written code and optimized images helped us create such a fast theme. Your visitors will love how fast your website is.

No-code admin panel

The Muffin Options panel allows users to customize the theme in an easy and simple way. There’s no need to code anything thanks to all the fantastic options that are included in the theme.

200+ shortcodes

The shortcode generator is what you need. The shortcodes allow you to build pages easily and quickly. Any element you see in the demo can be inserted anywhere on the page.

Child theme

Child theme files are included in the package, so you don’t need to create them yourself. All you need to do to get the child theme is to install it the same way as the parent theme.

Gorgeous Design

Parallax effect and video background

Create awesome pages with the smooth parallax effect that everyone loves. You can upload videos, put text, images, buttons, or other graphical elements, and enjoy the great looking sections.

Typography

Use Google or upload custom fonts, set sizes, style, weight, and much more. Be gives you full control over the fonts.

Unlimited colors

With the built-in color pickers, you can quickly change the color of nearly every element from the demo. Set your colors or use one of the pre-defined color skins.

Responsive and retina ready

Betheme is fully responsive and ready for retina displays. Your customers will love how your website looks on tablets and mobile devices.

Easy layout

Elementor ready

Betheme is fully compatible with Elementor with 30+ unique elements and dedicated pre-built websites.

Built-in mega menu

Our built-in mega menu is the perfect choice for large menus. You can set up columns and rows, use icons, and upload background images easily.

Wide or boxed layout

Choose between boxed and wide layout. This can be set globally for all pages or per single pages and posts.

One page

Betheme allows you to create beautiful one-page sites. It is an excellent opportunity for those who need simple websites with a smooth scroll effect.

Layouts configurator

Build custom layouts and assign them easily for different pages. The unlimited number of layouts gives you the possibility to create an unlimited amount of different looking pages on the same website.

Sidebars & widgets

Use single, dual or no sidebars on any post or page. Control the sidebar width, position and much more. You can also place specially-created widgets inside the sidebar, in the footer columns or even in the content.
